# 🤖 Persian AI Chat

**نسخه ۲.۰.۰** — اپلیکیشن اندروید چت هوش مصنوعی فارسی با طراحی مدرن

<div dir="rtl">

## ✨ ویژگی‌ها

- 🎨 **طراحی مدرن و حرفه‌ای** — الهام‌گرفته از ChatGPT با پشتیبانی از **حالت تیره و روشن**
- 🇮🇷 **راست‌به‌چپ کامل** — پشتیبانی کامل از زبان فارسی
- 💬 **حباب‌های چت انیمیشنی** — نمایش آواتار، زمان، دکمه کپی
- 📝 **ذخیره خودکار تاریخچه** — پیام‌ها با بستن برنامه حذف نمی‌شوند
- 🔌 **سازگار با OpenAI و APIهای مشابه** — DeepSeek، Groq و...
- ⚙️ **تنظیمات ساده** — API Key، Base URL و Model
- 🔄 **چک آپدیت درون‌برنامه‌ای**
- 📦 **CI/CD با GitHub Actions**

## 🚀 نحوه استفاده

۱. APK را از [Releases](https://github.com/salarzudfekr/persian-ai-chat/releases) نصب کنید
۲. از منوی سه‌نقطه بالا ⚙️ تنظیمات را باز کنید
۳. اطلاعات اتصال را وارد کنید
۴. شروع به چت کنید!

## 🛠️ سرویس‌های پشتیبانی‌شده

| سرویس | Base URL |
|-------|----------|
| OpenAI | `https://api.openai.com/v1` |
| DeepSeek | `https://api.deepseek.com/v1` |
| Groq | `https://api.groq.com/openai/v1` |
| OpenRouter | `https://openrouter.ai/api/v1` |

## 🏗️ ساخت از سورس

```bash
git clone https://github.com/salarzudfekr/persian-ai-chat.git
cd persian-ai-chat
./gradlew assembleDebug
```

نیازمندی‌ها: Android SDK 35، JDK 17، Gradle 8.9
